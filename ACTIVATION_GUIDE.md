## 💡 Pro Tips

1. **Always start metronome first** [M]
   → You'll stay in time automatically

2. **Hold chords for arpeggiator** [A]
   → Different chord shapes = different patterns

3. **Use effects subtly** [F]
   → They enhance, not replace your sound

4. **Check analysis** [X]
   → Learn your playing habits

5. **Try different tempos** [+/-]
   → Slow (80 bpm) for practice, normal (120) for sessions

## 📞 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| Metronome silent | Check Yamaha volume, confirm [M] pressed |
| Arpeggiator not working | Make sure [A] is enabled, hold a chord |
| Waveform not showing | Record something first, then [W] |
| Playback silent | Verify you recorded, check volume |
| App won't launch | Check Python installed, run `python --version` |
| Yamaha not detected | Confirm MIDI connection, run `check_midi.bat` |
