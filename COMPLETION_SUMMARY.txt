## ✨ First Session Walkthrough

**Step 1: Launch**
```bash
python studio.py
```
*Application starts, displays piano keyboard and controls*

**Step 2: Record**
- Press [R]
- Display shows: 🔴 RECORDING
- Play notes on your Yamaha

**Step 3: Analysis (Real-Time)**
- As you play, you see:
  - "Chord: C Major"
  - "Scale: Major"
  - Active notes highlighted
  - Velocity displayed

**Step 4: Stop**
- Press [T]
- Recording saved in memory

**Step 5: Save**
- Press [S]
- Session saved to: sessions/ folder

**Step 6: Export**
- Press [E]
- MIDI file saved to: exports/ folder
- Can now open in DAW
